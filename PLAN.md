## 问题 Statement
重新设计电商网站，展示每集视频的游戏卡片及电子版说明，用户购买后获得百度网盘地址及密码。

## 产品结构
- 16集视频 (G101-G112, G201-G212)
- G1系列: G101-G112 (12集) - 99.9元
- G2系列: G201-G212 (12集) - 99.9元  
- 单集: 9.9元

## 设计风格
- 皮克斯风格 (Pixar-style)
- 色彩明亮、温暖
- 卡通角色形象
- 圆润边角
- 活泼动效

## 推荐 Approach: Next.js + 简约架构

### 核心页面
1. **首页** - 游戏卡片网格展示 (G1/G2系列 + 单集)
2. **产品详情页** - 电子版说明 + 购买选项
3. **购物车/结算页** - 支付流程
4. **管理员后台** - 上传/管理百度网盘链接 (需登录)

### 技术方案
- Next.js 16 (App Router)
- Tailwind CSS + 自定义皮克斯主题
- localStorage 存储管理员凭据和网盘链接 (开发阶段)
- 模拟支付流程 (展示网盘链接)
